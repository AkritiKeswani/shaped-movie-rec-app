# Firebase Configuration
# Copy these values to Vercel Environment Variables

NEXT_PUBLIC_FIREBASE_API_KEY=your_firebase_api_key_here
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_project_id.firebaseapp.com
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id_here
NEXT_PUBLIC_FIREBASE_APP_ID=your_firebase_app_id_here
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_messaging_sender_id_here
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_project_id.appspot.com

# Instructions:
# 1. Replace all values with your actual Firebase project configuration
# 2. Add these same environment variables to Vercel:
#    - Go to Vercel Dashboard → Your Project → Settings → Environment Variables
#    - Add each variable with the exact same name and value
# 3. Redeploy your app after adding the environment variables
# 4. Make sure to add your Vercel domain to Firebase Auth → Settings → Authorized Domains
